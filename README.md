# 🎫 티케팅 연습 사이트

한국 티케팅 사이트(인터파크 등)에서 빠른 예매를 위한 클릭 속도 훈련 웹사이트입니다.

## 🌟 주요 기능

- **타겟 좌석 시스템**: 매번 예매해야 할 특정 좌석 번호가 지정됩니다 (예: 44번)
- **랜덤 좌석 생성**: 150개 좌석 중 8-15개가 랜덤으로 예매 가능하게 열립니다 (타겟 좌석 포함)
- **정확한 좌석 선택**: 지정된 좌석만 선택해야 예매 가능 (다른 좌석 선택 시 예매 불가)
- **반응속도 측정**: 좌석이 나타난 후 올바른 좌석을 찾아 예매 완료까지의 시간 측정
- **현실적인 대기순서**: 인기 가수 콘서트처럼 조금만 늦어도 10,000~20,000번대로 밀려남
- **성능 추적 그래프**: 연습할수록 실력이 얼마나 향상되는지 시각적으로 확인
- **통계 저장**: 브라우저 세션이 유지되는 동안 모든 기록이 자동 저장

## 🎮 사용 방법

1. "예매 대기중..." 화면에서 대기합니다
2. 좌석이 나타나면 **지정된 타겟 좌석 번호**를 확인합니다 (예: 44번)
3. 예매 가능한 좌석(분홍색) 중에서 **타겟 좌석 번호**를 빠르게 찾아 클릭합니다
4. 올바른 좌석을 선택하면 "예매하기" 버튼이 활성화됩니다
5. "예매하기" 버튼을 클릭하여 예매를 완료합니다
6. 결과를 확인하고 "다시 연습하기"로 반복합니다

### ⚠️ 주의사항

- **잘못된 좌석 선택 시**: 예매하기 버튼이 비활성화됩니다
- **좌석 재선택**: 잘못 선택한 좌석을 다시 클릭하면 선택 취소됩니다
- **타겟 좌석만**: 지정된 번호의 좌석만 선택해야 예매 가능합니다

### 💡 팁

- 타겟 좌석 번호를 먼저 확인하세요
- 좌석 배치를 빠르게 스캔하는 연습을 하세요
- 모든 예매 가능한 좌석이 같은 색이므로 번호를 정확히 읽어야 합니다
- 손가락을 마우스 위에 미리 올려두세요
- 긴장을 풀고 자연스럽게 클릭하세요
- 키보드 단축키: 스페이스바 또는 엔터키로도 예매 가능합니다

## 📊 통계 설명

- **시도 횟수**: 총 연습 횟수
- **평균 반응속도**: 전체 시도의 평균 반응 시간 (ms)
- **최고 기록**: 가장 빠른 반응 시간
- **평균 대기순서**: 평균적으로 받은 대기순서 번호



## 🎯 대기순서 계산 방식 (현실적인 인기 콘서트 기준)

반응속도에 따라 대기순서가 결정됩니다:

| 반응속도 | 대기순서 범위 | 평가 |
|---------|-------------|------|
| 0-200ms | 1 ~ 200번 | 🔥 매우 빠름 - 티켓팅 성공 가능! |
| 200-500ms | 200 ~ 700번 | 🚀 빠름 - 성공 가능성 있음 |
| 500-1000ms | 700 ~ 1,500번 | 💪 보통 - 연습 필요 |
| 1000ms 이상 | 대략 1.5:1 비율 | ⚡ 느림 - 더 빠르게! |

**예시**: 
- 1000ms → 약 1,500번
- 2000ms → 약 3,000번
- 2722ms → 약 4,000번

빠르게 찾을수록 앞순서를 받을 수 있습니다!

```javascript
localStorage.removeItem('ticketingPracticeStats');
location.reload();
```
